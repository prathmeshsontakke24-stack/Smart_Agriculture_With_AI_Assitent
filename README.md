# 🌱 Smart Agriculture & AI Assistant

## 📌 Project Overview

Smart Agriculture & AI Assistant is an intelligent agriculture management system developed to support modern farming using Artificial Intelligence and web technologies. The main objective of this project is to help farmers improve productivity, monitor crops efficiently, reduce manual effort, and make better farming decisions using AI-powered recommendations.

The system provides a smart platform where farmers can manage crop details, receive weather information, monitor farming activities, and communicate with an AI assistant for agriculture-related guidance. The AI assistant is integrated using the Google Gemini API, which helps users by answering farming questions, suggesting fertilizers, identifying crop-related problems, and providing smart farming techniques.

This project is developed using Java Spring Boot for backend development, MongoDB as the database, and HTML, CSS, and JavaScript for frontend user interface design.

# ⚙️ Working of the System

1. Farmers register and log into the system.
2. Farmers add crop and farming details.
3. The system stores all data in MongoDB database.
4. Weather data is displayed to help farming decisions.
5. Farmers ask agriculture-related questions to the AI assistant.
6. Gemini API processes the query and provides intelligent responses.
7. The system displays AI recommendations and farming suggestions.
8. Farmers use these suggestions to improve crop productivity and management.
